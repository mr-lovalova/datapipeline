# Jerry Thomas demo

This project is generated by `jerry demo init` and contains a complete local
dataset with source, stream, feature, artifact, and serve configurations.

```bash
python -m pip install -e .
jerry serve --dataset demo --limit 3
```

The bounded NumPy model-batch example lives at
`demo/scripts/run_model_batches.py`. Install `jerry-thomas[ml]` before running
it.
